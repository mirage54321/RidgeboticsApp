My name is Mira and this is my robotics app!
This app is made with AI scanning so that you can find any errors in your robot (based off of photos).
This app has two main parts right now:
* Scanning for visible errors
* Scanning to see if it follows FRC rulebook

I made this app because I am on a robotics team with limited people and I believe that this will help teams find problems in physical aspects. I believe that (when finished) this app can reduce the time it takes to locate the error so that teams have more time to fix the error.

I also made this app so that I could work with AI before college!

AI usage: I implemented Gemini to scan the images. I used Claude to help debug and help me learn how to write lines of code that I wasn't sure how to write. Especially with the UI because Flutter is very difficult to work with.

To look at the app go to this link: https://mirage54321.github.io/RidgeboticsApp/

After that it is pretty straight forward! Just select the part you want to check your robot for, upload a photo, click scan, and just wait for the AI to find problems!
